Scroll service

All directives in this package rely on the Scroll service. It's a simple service that makes sure only a single event listener is active at a time.

This saves some performance if you consider the rapid-fire scroll event.

Other modules can attach itself to the service if they want to be called on scroll.

Usage

angular.module('myApp', ['grrrScroll'])
.directive('myDirective', function(Scroll) {
  return {
    link: function(scope, element, attrs) {
      Scroll.attach('myIdentifier', function() {
        // your scroll event here
      });
    }
  };
});

Use detach() when you're done listening to the scroll event:

Scroll.detach('myIdentifier');
